# Sahaya-women-safety-app
A Flutter-based mobile application designed to improve women’s safety by providing emergency SOS features, real-time location sharing, and access to safety resources.

Overview

Sahaya is a safety-focused mobile application that helps users respond quickly during emergency situations. It allows users to send SOS alerts, share their live location, and access important safety tools and information.

Key Features

Emergency SOS

Shake device to trigger SOS alert by shaking 2–3 times Sends emergency message with live location to trusted contacts Supports up to 3 emergency contacts

Location Sharing

Share real-time location with emergency contacts Integrated map for location tracking

Safety Tools

View nearby police stations and safe zones Fake call feature for uncomfortable situations Quick access to emergency helpline numbers Safety tips and self-defense information

Community Features

Create and share posts Comment and interact with other users Community-based support system

Admin Panel

Manage safety tips and articles Moderate community posts Tech Stack

Flutter – Mobile application framework Dart – Programming language Supabase – Backend and authentication OpenStreetMap – Map integration Geolocator – Location services

Purpose

The purpose of Sahaya is to provide a quick-response safety tool that helps users feel more secure and supported during emergency situations.
